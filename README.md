## [TIL] - String Format Specifiers
- **정수형**
    - %d, %u (unsigned)
    - %ld, %lu (long 타입)
    
- **실수형**
    - %f, %e (지수형)
    
- **16진수, 8진수**
    - %x, %o
    
- **객체형**
    - %@

> **[Official Docs]**: [Format Specifiers](https://developer.apple.com/library/archive/documentation/Cocoa/Conceptual/Strings/Articles/formatSpecifiers.html)
